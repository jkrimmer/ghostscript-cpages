PDF Pagecolor Analyzer

Select a PDF file to obtain the number of blank, black and white, and color pages of the document. The selected PDF file does not leave your device, all the processing happens locally in your browser. This way, we can ensure data privacy.

Depending on your file, the processing might take some time and your browser might seem stuck. If this happens, please be patient!

This webapp is enabled by Ghostscript, which I have compiled to WebAssembly using Emscripten. Since Ghostscript is distributed under the Affero General Public License, the source code of this webapp is available under the same license on GitHub ↗.
